VIT Academic Components

Software Development Project (SDP)

SDP focuses on building real-world software systems using agile methodologies, teamwork, and version control.

Engineering Design and Innovation (EDI)

EDI emphasizes problem solving, innovation, and design thinking to create impactful engineering solutions.

Design Thinking (DT)

DT teaches user-centric design, ideation, prototyping, and iterative problem solving.

Course Projects

Course projects reinforce theoretical knowledge through practical implementation and problem-solving.
